# Minor Illusion

For this 48-hour [Micro Jam](https://itch.io/jam/micro-jam-057) with the theme “Symmetry” and the additional requirement “Mirrored Mechanics”, me and [Haroon](https://theroonguy.itch.io/) wanted to avoid the obvious ideas (gravity inverters, controlling two mirrored characters, etc.) – and decided on a tower defense game where you control a wizard that can cast mirror spells, which allow him to place mirrored duplicates of his magic defense tower, so that they can shoot the approaching enemy bugs. As waves of enemies approach the main tower, the player has to create as many duplicates as possible in order to survive.

While I handled all of the programming and Unity engine stuff for this jam, Haroon was pumping out all of the fantastic pixel art, which makes this little game really charming.
