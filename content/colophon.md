# Colophon

This website was created using [Hugo](https://gohugo.io/) as a static site generator, for which I've created a custom theme (the current theme being particularly influenced by [Cosma's](https://bactra.org/) minimalist/early-internet homepage, as well as browsing through various [archived geocities sites](https://geocities.restorativland.org/Athens/Atlantis/1125/)).

My intention was to source as much content has possible directly from my personal [Obsidian](https://obsidian.md/) vault, so I wrote a simple Python script for exactly that (especially with the [notes]({{< ref "notebook" >}}) section).

I'm hosting the site for free with [GitHub Pages](https://docs.github.com/en/pages), connected to my own domain. The public GitHub repository can be found [here](https://github.com/paulinprogress/Website).
