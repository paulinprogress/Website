# Small Web

Der Gegensatz zum kommerziellen [[Internet|Web]], das von großen [[Technologie|Tech]]-Konzernen dominiert wird, und das man primär als “User” bzw. Kunde navigiert.

Das Small Web ist dagegen [[Dezentralisierung|dezentral]] und besteht aus persönlichen Blogs, nischigen Hobby-Projekten und [[Digital Garden|Digital Gardens]], die einen großen Teil des frühen [[Internet|Internets]] ausmachten ([[1990s]]/[[2000s]]):

- [[Geocities]]: Ein benutzerfreundlicher Service, um eigene Webseiten zu erstellen, der von 1994 bis 2009 aktiv war.
- [[Webrings]]: Thematisch verwandte Webseiten, die via [[Hyperlinks]] miteinander verbunden waren.
- (usw.)

Dabei bezieht sich das Small Web heute, zumindest wie es 2020 von Aral Balkan definiert wurde, aktiv auf “[[Small Technology]]”-Prinzipien:
