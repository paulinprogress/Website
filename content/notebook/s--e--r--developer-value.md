# S × E × R = Developer Value

[[Heuristik]] zur Einschätzung eines Teammitglieds (hier im [[Game Development]], aber ist auf vieles übertragbar), nach [[Jesse Schell]]:

- S = Skill (siehe: [[Leistung]], [[Effektivität]], etc.)
- E = Enthusiasmus (siehe: [[Motivation]], [[Lernen]], etc.)
- R = “Respectabilitude” – die Fähigkeit, anderen das Gefühl zu geben, respektiert zu werden (siehe: [[Empathie]], [[Kommunikation]], etc.)

Mit dem entscheidenden Hinweis: 100 x 100 x 0 = 0 …
